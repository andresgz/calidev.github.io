Agenda

Conoce los contenidos que se van a compartir en el CaliDev.

  • 10:00 a.m.

    Introducción y Bienvenida

    Lanzamiento de CaliDev

    Explicaremos todo acerca de la comunidad: el propósito, los objetivos y el compromiso con el desarrollo de la ciudad.

  • 11:00 a.m.

    Diseño, como la vida misma

    UX/UI, Web Design

    Por Julian Mora

  • 12:00 p.m.

    Lunch Time!

    Lunch Deployment

    Let the hunger games begin!

  • 1:00 p.m.

    El qué, cómo y para qué de Ionic Framework

    AngularJS, Apache Cordova, SASS

    Una breve introducción al desarrollo de aplicaciones móviles híbridas haciendo énfasis en el uso de Ionic como herramienta principal en nuestros proyectos.

    Por Richard Melo

  • 2:00 p.m.

    Una Simple Web API en Ruby

    Ruby/Rails/API's

    Esta es una charla sobre cómo construir una Web API básica, y que herramientas existen en el ecosistema de Ruby para lograrlo.
    I Love Ruby! - Ruby On Rails - Sinatra

    Por Moises Vargas

  • 3:00 p.m.

    Indices Geoespaciales con MongoDB

    MongoDB/NodeJS/API's

    Expondremos cómo hacer una API Básica con búsqueda geoespacial con MongoDB y NodeJS utilizando como ejemplo el user story de "Buscar eventos cercanos a menos de 500 metros".

    Por Julian Cantillo

  • 4:00 p.m.

    Cali vs World

    Exporting Services

    Exporting Digital Services Internationally: Opportunities, Challenges, Risks and Best Practices.

    Por John Coffey

  • 5:00 p.m.

    Despliegue de aplicaciones para alta disponibilidad

    Consideraciones para llevar una aplicación al siguiente nivel. Consideraciones de diseño, configuración de aplicaciones en el servidor, herramientas para el despliegue y monitoreo de nuestras aplicaciones.

    Por Andres Gonzalez

  • 6:00 p.m.

    Data Science Essentials

    Machine Learning, PySpark, MLLib y Apache Spark.

    En esta charla se cubrirán los elementos esenciales en Data Science. Desde la base academica, pasando por el estado del arte en herramientas para Data Science hasta un ejemplo de Machine Learning con MLLib (PySpark) de Apache Spark

    Por Daniel Valencia
